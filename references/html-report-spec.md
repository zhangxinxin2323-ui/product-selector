# HTML 决策卡

决策卡用于快速判断，不替代完整报告。

## 区块

- 市场名、ASIN、日期和综合状态
- Market Decision、Financial Decision 与 Launch Feasibility 三个独立状态
- 8 个关键 KPI
- 3-5 条关键发现
- 风险、hard gates 和缓解方案
- 反向总落地成本上限；有头程依据时再显示采购价上限，或展示正式财务结论、峰值资金与回本月
- 下一步最小实验
- 来源和完整报告路径

移动端保持两列 KPI，文本不得溢出。使用绿、橙、红表达通过、条件和失败，但不要只靠颜色传达状态。

文件写到当前 run directory，不硬编码用户主目录。
