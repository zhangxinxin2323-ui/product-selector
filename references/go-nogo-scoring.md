# Go/No-Go 评分体系

> 所有正式分析必须输出 Go/No-Go 评分卡。即使用户已经倾向进入，也必须把风险和前提显式写出来。

## 五维加权评分

| 维度 | 权重 | 评分标准（1-10） |
|------|------|-----------------|
| **市场规模** | 20% | 基于月搜索量、类目月销量、增长趋势 |
| **竞争格局** | 25% | 基于品牌集中度、Top3垄断率、新品占比 |
| **需求清晰度** | 15% | 基于关键词多维覆盖、VOC痛点明确度 |
| **进入壁垒（反向）** | 20% | Review壁垒+资金壁垒+技术壁垒+合规壁垒+供应链壁垒+品牌壁垒 |
| **盈利能力** | 20% | 基于毛利率、M值、广告空间、盈亏平衡CPC |

## 各维度评分细则

### 市场规模 (20%)

| 分数 | 条件 |
|------|------|
| 9-10 | 核心词搜索量 >100K/月，类目月销 >$5M，强增长趋势 |
| 7-8 | 搜索量 50-100K，类目月销 $1-5M，稳定增长 |
| 5-6 | 搜索量 20-50K，类目月销 $500K-1M，平稳 |
| 3-4 | 搜索量 10-20K，类目月销 $100-500K，增速放缓 |
| 1-2 | 搜索量 <10K，类目月销 <$100K，衰退 |

### 竞争格局 (25%)

| 分数 | 条件 |
|------|------|
| 9-10 | Top3垄断率 <20%，品牌数 >100，新品占比 >30% |
| 7-8 | 垄断率 20-40%，品牌数 50-100，新品占比 15-30% |
| 5-6 | 垄断率 40-60%，品牌数 30-50，新品占比 10-15% |
| 3-4 | 垄断率 60-80%，品牌数 10-30，新品占比 5-10% |
| 1-2 | 垄断率 >80%，品牌数 <10，新品占比 <5% |

### 需求清晰度 (15%)

| 分数 | 条件 |
|------|------|
| 9-10 | 3+维度关键词都有高搜索量，VOC痛点集中在2-3个可解决方向 |
| 7-8 | 2-3维度覆盖，痛点明确但方向较多 |
| 5-6 | 仅核心词有量，延伸词分散，痛点不够集中 |
| 3-4 | 关键词混池严重，难以界定竞争单元 |
| 1-2 | 需求模糊，无法确定目标市场 |

### 进入壁垒 (20%，反向评分，壁垒越低分越高)

| 分数 | 条件 |
|------|------|
| 9-10 | 无认证要求，Review门槛<100，首批<$3K，成熟供应链 |
| 7-8 | 基础认证（1-2个），Review<300，首批$3-10K |
| 5-6 | 需要多个认证，Review<500，首批$10-30K |
| 3-4 | 复杂认证（FDA等），Review>500，首批$30-100K |
| 1-2 | 高度管制品类，Review>1000，首批>$100K |

### 盈利能力 (20%)

| 分数 | 条件 |
|------|------|
| 9-10 | 毛利率>40%，M值>2.5，广告空间>40%，盈亏平衡CPC远高于市场CPC |
| 7-8 | 毛利率30-40%，M值2.0-2.5，广告空间30-40% |
| 5-6 | 毛利率20-30%，M值1.6-2.0，广告空间20-30% |
| 3-4 | 毛利率10-20%，M值1.3-1.6，广告空间10-20% |
| 1-2 | 毛利率<10%，M值<1.3，广告空间<10% |

## 决策矩阵

| 加权总分 | 决策 | 行动 |
|----------|------|------|
| 7.5-10 | **GO** | 强烈建议进入，优先推进 |
| 6.0-7.4 | **CONDITIONAL GO** | 有条件进入，需解决关键风险后推进 |
| 4.0-5.9 | **HOLD** | 暂缓，需更多数据验证或等待时机 |
| 0-3.9 | **NO-GO** | 不建议进入，风险大于机会 |

```markdown

| 维度 | 权重 | 评分 | 依据 |
|------|------|------|------|

| 竞争格局 | 25% | X/10 | [一句话依据] |
| 需求清晰度 | 15% | X/10 | [一句话依据] |
| 进入壁垒 | 20% | X/10 | [一句话依据] |
| 盈利能力 | 20% | X/10 | [一句话依据] |
| **加权总分** | 100% | **X.X/10** | — |

**决策**：[GO / CONDITIONAL GO / HOLD / NO-GO]

**Top3 机会**：
1. [具体机会]
2. [具体机会]
3. [具体机会]

**Top3 风险 + 缓解方案**：
1. [风险] → [缓解方案]

```

**注意**：如果决策为 NO-GO，流程终止，不进入后续步骤。
